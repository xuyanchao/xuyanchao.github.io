作用域

#11.27学习记录

  • 变量作用域
    在JS中有两种变量，一种是局部变量，另一种全局变量。一般var定义的均为局部变量，只有在当前的作用与可以调用，不用var定义的变量一般为全局变量，在函数function中的var局部变量只能在函数内部使用，外面的用不了
  • 函数 函数有三种，函数表达式，声明函数，匿名函数，其中函数表达式可以在后面加上（），表示立即执行，但是声明函数却不可以这样，匿名函数可以在前面加上！ + - 等符号转化为函数表达式（可能我理解的不正确）在后面加上（）既可以立即执行，但是这种方式一般不推荐使用，最常用的还是（function（））（）这种写法最为正确，可以立即执行
function fn(){
var max = 10;
var fn2 = function(num){
     if(num>max){
         console.log(max)
         console.log(num)
         }
     }
     return fn2
     }
     var news = fn();
     news(20);

这段代码就是一个很绕的例子，其实下面的news就是执行过后的fn，fn有一个返回值return fn2 那么其实news就是fn2，news（20）就是fn2（20），那么num = 20，在控制台我们就会得到10和20两个数值。

前端的第一个月

初学web前端的那些乱七八糟的事

不知不觉来这学习已经一个月了，感觉真的好快，前几天韩哥突然问当时大家都是为什么来学H5的，突然就想起当时的我，为什么来，我的答案很简单，那就是混不下去了。

记得当时刚毕业的我在很短的时间内找了N份工作，都是干了几天就辞职了，甚至工资都没领，真正意义上的第一份工作竟然是一个不怎么能打的出台面的活，拿着少的可怜的工资，住在女朋友的住处，浑浑噩噩，整天就是闲的要死，整天在同学朋友的QQ群里插科打诨，斗图说笑，嘻嘻哈哈的，其实内心确实说不出来的愁苦····

干这一份什么都不是的工作混日子，自己感觉都说不过去，后来就在群里听见那个培训过的兄弟再说什么H5，我就问了问是什么东西，他们说的好多，但现在我几乎全都没印象，因为当时觉得我不是干这行的，也不打算干这行，就随便看了看，后来随着日子一天天过去，越开越缺钱花，才知道这样下去不行，想了想真的没什么出路了，看着人之前的同学都培训过，问了问他们，加上他们都不约而同的说H5不错，就毅然决然的辞了工作，带着另一个大学兄弟（也就是波哥）来到了这，其实当时想的真的很少，就是学门手艺，混口饭吃，当时心里最多的还是无奈，来到这我真的不知道H5是干什么的，虽然来之前百度了下，但是看着满屏幕的字，我发扬了之前的太长不看的精神，草草了事，直到第一节课才知道h5就是做前端的，前端就是web前段，也就是网站前端。
然后就是html,css至今已经接触了一点js，虽然每天的课都很满，但是心里却真的是很踏实···
什么都不想，做好自己当天的就行，我感觉现在想的再多也没用，做好当下的事就好了···
